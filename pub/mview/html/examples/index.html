  • PP2C superfamily   [new style sheet version].

    This is a hand finished CLUSTALW alignment. Some of the sequence identifiers (SWISS-PROT) are live links via SRS into the respective database. It was produced using built-in colour palettes and consensus pattern residue equivalence classes: member residues of the 80% consensus class for each column are coloured according to their own property.

    This particular alignment was published in Bork, P., Brown, N.P., Hegyi, H., Schultz, J. (1996) The protein phosphatase 2C (PP2C) superfamily: Detection of bacterial homologues. Protein Science. 5:1421-1425. MEDLINE.
